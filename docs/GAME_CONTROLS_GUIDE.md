## ðŸ› Troubleshooting

### **Common Issues:**

**Tractor won't move:**
- Make sure game is started (clicked "Start Farming!")
- Check WASD keys are working
- Fuel level > 0

**Crops not growing:**
- Advance time with "Next Day" button
- Check if seeds were planted (not just plowed)
- Ensure plot is watered

- Reduce browser zoom to 100%
- Close other tabs
- Clear browser cache
- Try Chrome for best performance

**Can't see crops:**
- Zoom in closer (scroll)
- Check plot info (click plot)
- Verify planting was successful
